# Source Directory Documentation

## Overview

The `src` directory contains the complete source code and technical documentation for the **Java JDBC Student Application**.

Unlike traditional repositories where only source code is available, this repository follows a documentation-first approach. Every important aspect of the project has been documented separately to help learners understand not only how the application works but also why each component is required.

The documentation inside this directory complements the root repository documentation and provides a deeper understanding of the application's implementation.

---

# Purpose

The purpose of the `src` directory is to:

- Store the Java source code.
- Explain the complete program implementation.
- Document JDBC concepts.
- Describe SQL operations.
- Explain application workflow.
- Provide execution guidance.
- Support interview preparation.
- Serve as a technical reference.
