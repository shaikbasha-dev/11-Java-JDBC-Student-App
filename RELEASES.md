# Current Stable Release

## Version 1.0.0

**Release Status:** Stable

### Highlights

- Java-based JDBC Student Application.
- Demonstrates MySQL database connectivity.
- Uses `PreparedStatement` for secure data insertion.
- Includes a simple DTO (`Demo`) class.
- Shows parameter binding for SQL execution.
- Demonstrates exception handling using `SQLException`.
- Performs manual cleanup of JDBC resources.
- Comprehensive project documentation for learners.
