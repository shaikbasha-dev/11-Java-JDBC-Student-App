# Resources

## Overview

The **11-Java-JDBC-Student-App** repository is designed as a practical learning resource for understanding Java Database Connectivity (JDBC) with MySQL. This document provides additional learning resources and recommendations that can help strengthen your knowledge beyond this project.

These resources are intended to support beginners, students, and interview candidates as they continue their Java and database programming journey.

---

# Prerequisite Knowledge

Before working with this project, it is recommended that learners have a basic understanding of:

- Java Programming Fundamentals
- Object-Oriented Programming (OOP)
- Variables and Data Types
- Methods and Classes
- Exception Handling
- Basic SQL Concepts
- Relational Databases
